# Personal Portfolio
A website that provides an overview of my professional identity as a graduating Human Computer Interaction student striving to enter the workforce.

## Sections
- **About**: Summary of my goals, achievements, and both academic and professional experience.
- **Experience**: Details of my professional experience.
- **Projects**: Cards featuring relevant imagery, brief descriptions, and links to deliverables for the projects I developed for competitions and college courses.
- **Education**: Details of my educational background.
- **Contact**: Contact information and professional socials.
- **Resume**: Hyperlink to a PDF of my resume.

## Features
- Fully responsive
- Valid HTML5 & CSS3
